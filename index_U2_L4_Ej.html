Programando el área del cuadrado

Área del cuadrado

 
            
Se calcula de la siguiente manera: área = lado * lado = lado2 

                En Javascript podemos declarar la siguiente función:
                
                    function area(lado){
                        return lado*lado;
                      }
                
                Si llamamos  a: area(4). Dará como resultado: 16
